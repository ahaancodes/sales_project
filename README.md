This project performs exploratory data analysis on sales data to understand revenue patterns, customer purchasing behavior, and product-level performance.

The workflow includes data preprocessing, statistical analysis, visualization of trends, correlation studies, and performance comparison across multiple variables. The project aims to uncover patterns that influence total sales and revenue growth.

Visual tools such as bar charts, scatter plots, and boxplots are used to interpret the dataset effectively. The analysis helps identify potential outliers and high-performing segments.

Technologies used: Python, Pandas, Matplotlib, Seaborn.
